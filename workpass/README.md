# WORK PASS 人材DB（Phase 1 MVP）

求職者データベース。基本情報〜価値観の登録に加え、**適性診断・一般常識テストをツール内で実行→自動採点**、面談評価の入力まで対応。

## 特徴
- **依存ゼロ・完全無料**：Node.js 標準機能のみ（内蔵SQLite）。`npm install` 不要。
- データはローカルの `workpass.db`（SQLite）に保存。

## 必要環境
- Node.js 22.5 以上（推奨 24+）。※内蔵SQLiteを使用。

## 起動方法
```bash
cd "採用＿ツール/workpass"
node server.js
```
起動後、ブラウザで **http://localhost:4000** を開く。
停止は `Ctrl + C`。ポート変更は `PORT=5000 node server.js`。

## 2つの入口
- **求職者向け（トップ `/`）**：本人が開いて登録する画面。ようこそ → プロフィール入力 → 適性診断 → 一般常識テスト → 完了、をガイド付きで進む（他人の情報や一覧は見えない）。
- **管理者向け（`/admin.html`）**：集まったデータの一覧・詳細・面談評価。

### 求職者フロー（本人が操作）
1. トップ `/` →「登録をはじめる」
2. プロフィール入力（基本／勤務条件／職歴／スキル／キャリア／価値観）
3. 適性診断（5段階回答 → 営業/接客/事務/マネジメント/コミュニケーションを0〜100で算出）
4. 一般常識テスト（5分野の選択式 → 自動採点）
5. 完了画面（担当者から連絡する旨を表示）

### 管理者フロー
- `/admin.html` の一覧 → 行クリックで詳細。詳細画面で**面談評価**（第一印象・コミュニケーション・積極性・人柄＋コメント）を入力。

## 収集項目（求職者DB）
- 基本情報：氏名／年齢／最寄駅／通勤可能範囲／連絡先
- 勤務条件：希望勤務地／勤務可能曜日／勤務可能時間／希望雇用形態／希望年収／希望月収／転職希望時期
- 職歴（複数可）：業界／職種／勤務年数／実績／退職理由
- スキル：営業／接客／事務／PCスキル／AIツール（0〜5）／保有資格
- 適性診断：営業／接客／事務／マネジメント／コミュニケーション（0〜100）
- 一般常識テスト：ビジネスマナー／敬語／国語／計算／ITリテラシー（0〜100）
- キャリア：希望職種／希望業界／3年後・5年後の目標／将来やりたい仕事
- 価値観：年収／成長／安定／人間関係／WLB（0〜5）
- 面談評価：第一印象／コミュニケーション／積極性／人柄／コメント

## 構成
```
workpass/
  server.js       HTTPサーバー＋API＋静的配信（依存ゼロ）
  db.js           内蔵SQLiteのスキーマ・クエリ
  questions.js    適性診断・一般常識テストの設問と採点ロジック
  public/         画面（一覧/登録/適性診断/常識テスト/詳細）
  workpass.db     データ本体（初回起動時に自動生成）
```
