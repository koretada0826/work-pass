# ビルド指示：企業リスト即時収集ツール【無料版】（WORK PASS 開拓モジュール）

## 目的
条件を入力すると該当企業を即座に収集し、「会社名・住所・電話番号・URL・業種」を
リスト化してCSV出力するWebツール。TELEMO架電リスト用。
★制約：APIキー課金・クレジットカード登録なしで動く「完全無料」構成で作る。
将来 自律エージェントが呼べるよう、収集ロジックは独立関数にする。

## 技術スタック
- Next.js 14 (App Router) + TypeScript
- 収集ロジックは `POST /api/prospect` のサーバーAPI（UIから分離）
- CSV出力（UTF-8 BOM付き＝Excelで文字化けしない）
- 外部有料APIキー・課金アカウントは使わない

## 入力（検索条件フォーム）
- 業種／カテゴリ（例：不動産, 飲食, 美容室, 建設 …）
- エリア（都道府県・市区町村、または中心地＋半径km）
- 取得件数上限（既定50・最大200）
- （任意）除外キーワード

## 収集元（すべて無料・キー不要）
1. **主：OpenStreetMap Overpass API**（無料・APIキー不要・クレカ不要）
   - 業種→OSMタグ（例 amenity/shop/office/craft）にマッピングしてエリア内を検索
   - 取得：name / 住所(addr:*) / website / phone・contact:phone / opening_hours
   - レート配慮：Overpassは連続叩きしない（1リクエストにまとめる／数秒間隔）
2. **従：企業サイトからの電話番号抽出**（無料）
   - 1で phone が空 & website がある企業のみ、そのサイトを取得し
     「会社概要/お問い合わせ」ページから電話番号を正規表現で抽出
   - robots.txt を尊重、1〜2秒/件のレート制限、User-Agent明示、失敗は握りつぶさずログ

## 出力（1社 = 1行）
| 会社名 | 業種/カテゴリ | 住所 | 電話番号 | URL | 電話の出所(OSM/サイト抽出/なし) | 取得日時 |
- 電話が取れない行も残す（"電話番号なし" と明記）
- 重複（同名＋同住所／同一電話）は自動デデュープ
- 画面：フォーム → 結果テーブル（ソート/絞り込み）→「CSVダウンロード」

## 挙動
- 送信 → `/api/prospect` → 取れた分から逐次テーブル表示 → CSV出力可
- タイムアウト/取得失敗は行単位でスキップし理由を記録

## コンプライアンス（必須）
- 対象は法人の公開ビジネス連絡先のみ。個人携帯・個人名義情報は収集・保存しない。
- robots.txt・各サイト規約を尊重。低レートで実行。
- リストは自社（Everエフォート/TELEMO）の営業架電用途に限定する旨をUIに明記。

## 成果物
1. 動作するWebツール（フォーム→収集→テーブル→CSV）
2. 独立関数 `searchCompanies(conditions): Promise<Company[]>`（後で自律エージェントが呼ぶ）
3. README（起動手順・無料である旨・コンプライアンス注記）
